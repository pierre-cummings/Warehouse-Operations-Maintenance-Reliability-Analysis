# Warehouse Operations Maintenance & Reliability Analysis

## Project Overview
Built a warehouse maintenance and reliability analysis project using Power BI and SQL-based analysis to evaluate downtime, repair cost, safety impact, and vendor versus in-house repair performance across multiple warehouse locations.

## Tools Used
- Power BI
- SQL
- Excel / CSV
- GitHub

## Key Focus Areas
- downtime by equipment
- maintenance cost by warehouse
- vendor vs in-house performance
- safety and operations impact
- reliability improvement opportunities

## Key Findings
- High-downtime equipment categories were identified as strong preventive maintenance targets.
- Vendor-supported repairs showed higher average downtime and cost than in-house repairs.
- Some warehouse locations carried higher maintenance cost and downtime burden than others.
- Operations-critical equipment showed the greatest business disruption when failures occurred.
